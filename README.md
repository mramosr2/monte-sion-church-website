# Monte Sión Church — Modern React Draft (ES/EN)

This is a **base / rough draft** React site inspired by https://www.misionmontesion.org/ but rebuilt with a clean, modern, accessible layout.

## Tech
- Vite + React
- Tailwind CSS
- React Router (HashRouter for GitHub Pages)
- i18next (Spanish/English toggle)

## Run locally
```bash
npm install
npm run dev
